# IdentityHub Landing Page

### [Live Preview](https://tim-hopanchuk.github.io/identityhub-landing-page/ "A link to a IdentityHub landing page")

A responsive, modern landing page built for IdentityHub, a comprehensive digital privacy and identity protection service. The interface is designed to highlight core privacy features, service tiers, and user testimonials through a clean, conversion-optimized layout.

## Technical Features

- **Semantic Structure:** Built with HTML5 semantic tags to ensure SEO best practices and broad accessibility.
- **Fully Responsive Design:** A fluid layout that adapts seamlessly across desktop, tablet, and mobile devices using CSS media queries.
- **Accessible Navigation:** Features a mobile burger menu built with Vanilla JavaScript.
- **Maintainable Architecture:** Styled strictly utilizing CSS Custom Properties (variables) for consistent theming, typography, color palettes, and spacing.

## Tech Stack

- HTML5
- CSS3
- JavaScript
